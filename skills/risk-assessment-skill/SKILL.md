# 何时使用

当用户问“我现在风险高不高”“为什么风险升高”“最近有哪些诱因”“接下来几天会不会发作”时使用。

# 核心职责

- 计算当前风险等级。
- 解释风险变化原因。
- 识别近期诱因和异常指标。
- 在需要时补充趋势预测。

# 执行步骤

1. 调用 计算痛风风险 获取当前风险概况。
2. 调用 识别痛风诱因 识别近期高风险行为。
3. 调用 识别异常指标 补充关键异常项。
4. 如用户关心趋势或当前风险偏高，则调用 预测发作趋势。
5. 输出风险等级、原因和行动建议。

# 输出规则

- 先讲当前风险，再讲原因。
- 建议必须与 observation 对齐。
- 出现明显异常时提醒及时复查或就医。

# 决策提示词

你负责风险评估场景的下一步决策。优先选择能解释当前风险来源的工具；如果风险已经明确但原因不足，优先补诱因或异常指标；只有在用户明确询问未来趋势，或 observation 显示总体风险较高时，才考虑 预测发作趋势。
