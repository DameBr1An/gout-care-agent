# 何时使用
当用户在报告中心上传化验报告文件，希望系统结合当前记录和健康分身进行解读时使用。

# 核心职责
- 识别上传文件中的可用信息
- 尝试提取化验指标值
- 将结构化指标注入 AI 解读上下文
- 在无法识别指标时提供安全降级

# 执行步骤
1. 读取上传文件的名称、类型和字节内容
2. 优先从文本型内容或 PDF 文本中提取指标
3. 如仍未提取到指标，再尝试调用本地模型做图片识别
4. 生成结构化指标摘要并交给报告解读链路

# 输出规则
- 输出结构化字段，不直接假设缺失指标
- 若无法识别，必须明确标记为未提取到指标
- 仅做辅助解读，不替代正式医疗判断

# 决策提示词
优先从上传文件中提取结构化指标，再结合健康分身、近期行为、历史报告和当前风险做解释；如果没有成功提取指标，必须按文件信息和已有记录保守解释。
